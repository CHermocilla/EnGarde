# En Garde!
A 2D medieval knights fighting game built with C# and MonoGame.

## About
En Garde! is a fighting game built around footsie mechanics, 
inspired by classic 2D fighters. Players control medieval knights 
in a battle of spacing, timing, and reads.

## Tech Stack
- C#
- MonoGame

## Features
- 2D sprite-based combat
- Footsie/neutral game mechanics
- Medieval knight characters and setting

## Development
Developed as a school project in 2024.
